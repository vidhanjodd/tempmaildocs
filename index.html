01 — Overview

What is TempMail?

A disposable email service that generates temporary inboxes that expire after 10 minutes. No signup required, no data retained.

📨

Instant Inbox

Generate a random temporary email address with a single click. No registration, no personal data required.

⏱

10-Minute Expiry

Every inbox automatically expires after 10 minutes. A scheduled cleanup job deletes all expired inboxes and emails.

🔒

Token-Based Access

Each inbox gets a unique 32-character secret token. Without it, no one can read your emails — prevents IDOR attacks.

🔄

Auto Refresh

Inbox page polls for new emails every 10 seconds automatically. No manual refresh needed.

🛡

XSS Protected

All email content is sanitized using DOMPurify before rendering. Malicious HTML/JS in email bodies cannot execute.

📬

Mailgun Webhook

Receives real emails via Mailgun's inbound routing webhook. Signature verification prevents fake webhook calls.

  id           UUID         PRIMARY KEY,
  email_address VARCHAR(255) UNIQUE NOT NULL,

CREATE TABLE emails (
  id          UUID    PRIMARY KEY,
  inbox_id    UUID    REFERENCES inbox(id),
  sender      VARCHAR(255),
  subject     VARCHAR(255),
  body        TEXT,
  received_at TIMESTAMP

Mailgun Webhook Signature Verification

Every inbound webhook is verified using HMAC-SHA256. The verification formula is:

signature = HMAC-SHA256(signingKey, timestamp + token)
if (computed_hex != signature) → return 200 "ignored"

07 — Key Code

Core Implementation

The most important pieces of the codebase explained.

InboxService.java — createInbox() Java 
@Service
@RequiredArgsConstructor
public class InboxService {

  private static final int EXPIRY_MINUTES = 10;

  public InboxDto createInbox() {
    String email = generateRandomEmail();   // 10-char alphanumeric
    String token = generateAccessToken();   // 32-char secret

    Inbox inbox = Inbox.builder()
      .emailAddress(email)
      .accessToken(token)
      .createdAt(LocalDateTime.now())
      .expiryTime(LocalDateTime.now().plusMinutes(EXPIRY_MINUTES))
      .build();

    return InboxMapper.toDto(inboxRepository.save(inbox));
  }

  private String randomString(int length) {
    String chars = "abcdefghijklmnopqrstuvwxyz0123456789";
    SecureRandom rng = new SecureRandom();
    StringBuilder sb = new StringBuilder(length);
    for (int i = 0; i < length; i++)
      sb.append(chars.charAt(rng.nextInt(chars.length())));
    return sb.toString();
  }
}
MailgunSignatureVerifier.java Java 
@Service
public class MailgunSignatureVerifier {

  @Value("${mailgun.webhook.signing-key}")
  private String signingKey;

  public boolean isValid(String timestamp, String token, String signature) {
    try {
      String data = timestamp + token;
      Mac mac = Mac.getInstance("HmacSHA256");
      mac.init(new SecretKeySpec(
        signingKey.getBytes(StandardCharsets.UTF_8), "HmacSHA256"));
      byte[] hash = mac.doFinal(data.getBytes(StandardCharsets.UTF_8));

      // Convert to hex and compare
      StringBuilder hex = new StringBuilder();
      for (byte b : hash)
        hex.append(String.format("%02x", b));

      return hex.toString().equals(signature);
    } catch (Exception e) {
      log.error("Signature verification error", e);
      return false;
    }
  }
}
CleanupService.java — deleteExpiredInboxes() Java 
@Slf4j
@Service
@RequiredArgsConstructor
public class CleanupService {

  @Transactional
  public void deleteExpiredInboxes() {
    List<Inbox> expired =
      inboxRepository.findAllExpiredBefore(LocalDateTime.now());

    if (expired.isEmpty()) return;

    // Step 1: delete emails first (FK constraint)
    for (Inbox inbox : expired)
      emailRepository.deleteByInboxId(inbox.getId());

    // Step 2: now safe to delete inboxes
    inboxRepository.deleteAll(expired);

    log.info("Cleanup complete — deleted {} expired inboxes", expired.size());
  }
}
inbox.js — safe email rendering JavaScript 
emails.forEach(e => {
  const div = document.createElement("div");

  // ✅ textContent for all untrusted fields — never innerHTML
  const subject = document.createElement("h4");
  subject.textContent = e.subject;         // safe

  const from = document.createElement("p");
  from.textContent = "From: " + e.sender;  // safe

  const body = document.createElement("div");
  // DOMPurify sanitizes HTML email bodies
  body.innerHTML = DOMPurify.sanitize(e.body);

  div.append(subject, from, body);
  container.appendChild(div);
});
